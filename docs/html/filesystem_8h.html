◆ color_mode_t

enum color_mode_t

Colour mode datatype. Used to set the colour mode of a key.

Enumerator
no_override 

Applies the layer colour mode to the key (default).

const_color 

Changes colour to a constant value regardless of key pressed state.

pressed 

Changes colour if the key is pressed.

not_pressed 

Changes colour if the key is not pressed.

toggle 

Toggles colour on every key activation.

midi_bound 

Reacts to the current MIDI state bound to the key.

Only woks if MIDI CC command was bound for Key. Uses same channel and number as set in key.

disabled 

Disables the LED regardless of layer colour mode.
